# training-data-codeforces

This module is the Codeforces vertical OJ slice for `platform-training-data`.

It owns Codeforces ODS ingest, admin recent-lookback submission collection, admin student-data purge, Codeforces handle-account mapping, ODS/DWD/DWM/DWS schema migrations, idempotent SQL task resources, the SQL task manifest, admin warehouse refresh, internal Java read-side query services, public DWD/DWM/DWS query HTTP endpoints, the automatic-collection DWS summary list, local fixtures, and tests. Personal app-layer read queries use platform `studentIdentity` and resolve it to a Codeforces handle internally before querying warehouse repositories.

## Layer Rules

Dependency direction:

```text
web -> app -> domain
infra -> domain
config -> app/domain/infra/web
```

Rules:

- `domain` must not depend on `app`, `infra`, or `web`.
- `app` orchestrates use cases and should not build SQL.
- `infra` owns parsing, JDBC SQL, and database row mapping.
- `web` owns HTTP controllers and response DTOs only.
- `config` wires Spring beans and should not contain business logic.
- Codeforces warehouse time fields use UTC+8 (`Asia/Shanghai`) semantics.

## Directory Layout

```text
src/main/java/com/custacm/platform/trainingdata/codeforces/
  app/                 # application use cases; no HTTP or JDBC details
    account/           # Codeforces handle-account use cases and app errors
    collector/         # submission collection use case, collection jobs, and collection results
    ingest/            # ODS ingest use case and ingest results
    purge/             # student-scoped Codeforces data deletion use case
    query/             # DWD/DWM/DWS read use cases and query results
    warehouse/         # admin warehouse refresh use case
  collector/config/    # typed collector source/schedule properties
  scheduler/           # scheduled adapters that invoke app use cases
  domain/              # domain boundary; no dependency on app/infra/web
    model/             # ODS/DWD/DWM/DWS records and read models
    value/             # reusable value objects
    criteria/          # query criteria records and default filter helpers
    collector/         # external Codeforces source client port
    parser/            # parser ports consumed by app services
    repo/              # repository/writer interfaces
  infra/               # infrastructure implementations
    collector/         # external Codeforces HTTP clients
    parser/            # Codeforces payload parsing
    repo/              # JDBC repository/writer implementations and SQL mapping
  web/                 # HTTP layer
    account/           # handle-account controller, request DTOs, response DTOs, and error mapping
    collector/         # submission-collection controller, request DTOs, response DTOs, and error mapping
    ingest/            # ODS ingest controller and response DTOs
    purge/             # admin student-data purge controller and response DTOs
    query/             # public warehouse query controller and response DTOs
    warehouse/         # warehouse refresh controller, request DTO, and error mapping
  config/              # Spring Bean wiring

src/main/resources/
  db/migration/        # Flyway migrations owned by this OJ module
  fixtures/codeforces/ # stable local Codeforces test fixtures
  sql/ods/             # ODS upsert SQL
  sql/dwd/             # DWD set-based transform SQL
  sql/dwm/             # DWM set-based transform SQL
  sql/dws/             # DWS set-based transform SQL
  sql/tasks/           # SQL task DAG manifests

src/test/java/com/custacm/platform/trainingdata/codeforces/
  app/account/         # handle-account app tests
  app/collector/       # collection app and job-service tests
  app/purge/           # student-data purge app tests
  app/query/           # read-side query app tests
  app/warehouse/       # warehouse refresh app tests
  domain/model/        # model validation tests
  domain/criteria/     # criteria pass-through and default-helper tests
  infra/collector/     # external source client tests
  infra/parser/        # parser fixture tests
  infra/repo/          # JDBC repository and SQL task tests
  scheduler/           # scheduled adapter tests
  web/account/         # handle-account controller tests
  web/collector/       # collection controller tests
  web/ingest/          # ingest controller tests
  web/purge/           # student-data purge controller tests
  web/query/           # query controller tests
  web/warehouse/       # warehouse refresh controller tests
```

`target/` is generated by Maven and is not part of the maintained source layout.
